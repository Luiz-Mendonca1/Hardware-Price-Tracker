# Hardware Price Tracker & Predictor

Este projeto é um sistema de Data Pipeline focado em coletar, processar e analisar a flutuação de preços de hardware (foco inicial em GPUs) nos principais varejistas brasileiros. O objetivo final é alimentar um modelo de Machine Learning para prever tendências de baixa e identificar oportunidades reais de compra.

## Como Executar
Configure o ambiente virtual:
```
python -m venv venv
./venv/Scripts/activate
```
Instale as dependências:
```
pip install selenium beautifulsoup4 requests    
```
